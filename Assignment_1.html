Executive Summary

Used Decision Tree Approach to categorize the classe variable. Shortlisted the 6 most appropriate predictors from out of the 160 predictor variables. Got a 60% success rate on the test set.

Cross Validation

For this problem it is important to find the correct features that would help in developing an appropriate decision tree. So we divide our train data into train and test set. Now each time we check for the train train set the best input features using this 3 step approach 1. Remove all Near Zero Variance Predictors 2. Check for correlation between the classe and other variables and one with > 0.15 are selected 3. Removing variables like min, max, avg, amplitude, stddev and var

We take the union of final variables for each loop.
